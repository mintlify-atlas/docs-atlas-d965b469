Dify is an open-source platform for developing LLM-powered applications. It combines an intuitive visual interface with production-grade infrastructure to help teams go from prototype to deployment without needing deep ML expertise.

## Architecture overview

Dify has two main components:

- **Backend API** (`api/`) — A Python Flask application implementing Domain-Driven Design. It handles LLM orchestration, RAG indexing, agent execution, workflow runs, and exposes REST endpoints for both the web UI and external integrations.
- **Frontend** (`web/`) — A Next.js + TypeScript application providing the visual workflow builder, Prompt IDE, knowledge management UI, and the end-user chat interface.

Both components are packaged as Docker images and orchestrated together via Docker Compose or Kubernetes.
