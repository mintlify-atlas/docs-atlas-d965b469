## What is Dify?

Dify is an open-source LLM application development platform. Its intuitive interface combines AI workflow orchestration, Retrieval-Augmented Generation (RAG) pipelines, agent capabilities, model management, and production observability — letting you go from prototype to production quickly.

<CardGroup cols={3}>
  <Card title="Visual Workflows" icon="diagram-project">
    Build complex AI pipelines on a visual canvas with drag-and-drop nodes for LLMs, code, tools, and data.
  </Card>
  <Card title="RAG Pipeline" icon="database">
    Ingest documents (PDF, DOCX, PPT) into a knowledge base and connect it to any LLM with semantic search.
  </Card>
  <Card title="AI Agents" icon="robot">
    Create agents with ReAct or Function Calling strategies backed by 50+ built-in tools.
  </Card>
  <Card title="100+ LLM Providers" icon="microchip">
    Connect to OpenAI, Anthropic, Google, Mistral, Llama, and any OpenAI-compatible endpoint.
  </Card>
  <Card title="LLMOps" icon="chart-line">
    Monitor logs, annotate outputs, and continuously improve prompts with production data.
  </Card>
  <Card title="Backend-as-a-Service" icon="server">
    Every feature is backed by a REST API so you can embed Dify into any application.
  </Card>
</CardGroup>
